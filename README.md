## :rocket: Clonando o Projeto

```bash
# Clone this repository
$ git clone https://github.com/MogLuiz/solid-article.git

# Go into the repository
$ cd solid-article

# Install dependencies
$ yarn install

# Run the app
$ yarn start
```
